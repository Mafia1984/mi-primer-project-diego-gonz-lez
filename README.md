# Vue 3 + Vite + TypeScript + Pinia + Router + Axios

Proyecto base listo e inicializado en WebNative.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview


## WebNative (Fission)
- SDK instalado: `webnative` (API moderna basada en `program()`).
- Inicialización en: `src/webnative.js`.
- Invocación desde: `src/main.ts` (después de `app.mount()`).
- Evidencia en consola (dev):  
  - "WebNative inicializado correctamente"  
  - "Estado WebNative (program): { hasSession: false, username: null }"
```
